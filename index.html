Tech DBs Loader (Multi-Sheet)

🔑 طريقة الاستخدام بدون مشاكل:
اذهب إلى Supabase Dashboard → Edge Functions → api → Secrets

ثم ضع نفس الكلمة في حقل Admin Secret أدناه. لن تحتاج token بعد الآن.
مثال: https://<PROJECT_ID>.supabase.co/functions/v1/api
